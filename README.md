# Higa Collective

**Five minds, one origin.** — A tribute to the enduring friendship of five classmates from Higa Model Boarding School.

## Overview

Higa Collective is a modern, responsive website showcasing five friends and their diverse talents: software engineering, music production, piano, choir leadership, and worship. The site combines elegant design with smooth interactions to celebrate their bond and collaboration.

## Features

- **Team Cards** — Interactive member profiles with hover effects and modal popups
- **Contact Links** — Direct Gmail and Telegram access for each member
- **Responsive Design** — Optimized for desktop, tablet, and mobile
- **Modern UI** — Dark theme with orange accent, custom typography, and ambient glows
- **Navigation** — Smooth scroll behavior and mobile-friendly menu

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No build process or dependencies required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/higa-collective.git
cd higa-collective
```

2. Open `index (1).html` in your browser:
```bash
start "index (1).html"
```

Or use a local server (recommended):
```bash
python -m http.server 8000
# Then visit http://localhost:8000
```

## Usage

- **Click any member card** to open their contact modal with links to social profiles and email
- **Use the hamburger menu** (top-left) to navigate between sections
- **Scroll smoothly** to explore the full story
- **Tap Telegram buttons** at the bottom to reach members directly

## Technologies Used

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, flexbox, gradients, animations
- **JavaScript (Vanilla)** — Modal interactions, menu toggle, keyboard shortcuts
- **Fonts** — Bricolage Grotesque (display), Inter (body)

## Customization

### Colors
Edit the CSS variables at the top of `<style>`:
```css
:root {
  --bg: #060606;
  --accent: #ff7a1a;
  --text: #f5f0ea;
  /* ... more colors */
}
```

### Member Information
Update the card and modal sections in the HTML with new names, roles, photos, and links.

### Photos
Replace image file names in `<img src="">` tags with your own photo filenames.
